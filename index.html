Mobile Application for FAST Students

Joseph Akandwanaho
Mobile App Developer

The Orientation App is a user-friendly platform developed for the Faculty of Applied Sciences and Technology at MUST, aimed at enhancing the experience of students, staff, and administrators. It provides access to essential faculty information, including the dean's welcome message, mission, vision, and core values (Innovation, Professionalism, Integrity, Teamwork). The app features details on departments, programs, and staff, offering a centralized hub for exploring academic offerings and resources. With a secure login system and a responsive interface, it supports community engagement and supports the faculty's commitment to cutting-edge education and research.

Installation Instructions

  1. Click the download button above to download the APK file
  2. Go to your phone's Settings → Security → Unknown Sources and enable it
  3. Open your phone's file manager and locate the downloaded APK file
  4. Tap on the APK file and follow the installation prompts
  5. You may need to allow installation from your browser or file manager
  6. Once installed, you can find the app in your app drawer
